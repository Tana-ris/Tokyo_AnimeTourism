## 聖地巡礼アプリ「SEICHI」

[都知事杯オープンデータ・ハッカソン](https://odhackathon.metro.tokyo.lg.jp/)（2023年）にて，立正大学後藤研究室として参戦。  
アニメの聖地巡礼に着目し，外国人観光客向けに観光ルートを提案する ***『SEICHI』*** を提案。
